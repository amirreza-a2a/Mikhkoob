
# 🛡️ Mikhkoob (Focus Guardian)

**دستیار هوشمند کار عمیق** با تشخیص حضور با وب‌کم، مسدودساز سایت‌های مزاحم،  
زمان‌بند استراحت مبتنی بر مدل خستگی عصبی، و صفحهٔ «سپر تمرکز» برای بازگشت به مسیر درست.

---

## ✨ امکانات

- **تایمر فوکوس هوشمند** – جلسات مطالعه را با مدت دلخواه شروع کنید و برنامه حضور شما را با وب‌کم کنترل می‌کند
- **تشخیص ترک میز** – اگر از جلوی دوربین بروید، ابتدا هشدار و سپس جلسه را متوقف می‌کند
- **مسدودساز سایت** – افزونهٔ کروم/فایرفاکس سایت‌های حواس‌پرت‌کن را حتی در زمان استراحت هم مسدود نگه می‌دارد
- **صفحهٔ سپر تمرکز** – وقتی وسوسه شدید و سایت ممنوعه را باز کردید، یک صفحهٔ زیبا با تمرین تنفس، آمار و میانبرهای مفید شما را برمی‌گرداند
- **مدل خستگی عصبی** – بر اساس زمان فوکوس شما، زمان استراحت بهینه را محاسبه می‌کند (مثلاً ۴۵ دقیقه کار → ۶ دقیقه استراحت)
- **آمار و نمودارها** – عملکرد روزانه، هفتگی و Focus Score خود را در پنل گزارش ببینید
- **پروفایل‌های زمانی** – برای کارهای مختلف (مثلاً «درس عمومی» ۴۵ دقیقه، «درس تخصصی» ۷۵ دقیقه) پروفایل بسازید
- **بستهٔ نصبی دبیان (.deb)** – با یک دستور روی اوبونتو نصب می‌شود

---

## 📦 نصب

### نصب بستهٔ .deb (اوبونتو)
```bash
sudo dpkg -i focusguardian_1.0.0_amd64.deb
```

### اجرا از سورس (توسعه)
```bash
git clone https://github.com/amirreza-a2a/Mikhkoob.git
cd Mikhkoob
bash run.sh
```

### نصب افزونهٔ مرورگر
۱. مرورگر کروم (یا فایرفاکس) را باز کنید  
۲. به `chrome://extensions` بروید  
۳. حالت **Developer mode** را روشن کنید  
۴. روی **Load unpacked** کلیک کنید  
۵. پوشهٔ `extension/` را انتخاب کنید  

---

## 🧠 نحوهٔ استفاده

۱. **انتخاب زمان** – حلقهٔ تایمر را بچرخانید یا روی پروفایل کلیک کنید  
۲. **شروع فوکوس** – روی دکمهٔ «شروع فوکوس» کلیک کنید  
۳. **حضور** – جلوی وب‌کم بمانید. اگر بروید، ابتدا هشدار (۵ ثانیه) و سپس جلسه لغو می‌شود (۲۰ ثانیه)  
۴. **پایان جلسه** – تایمر که تمام شد، زمان استراحت پیشنهادی شروع می‌شود  
۵. **گزارش** – پنل سمت راست آمار روزانه، هفتگی و نمودارهای عملکرد را نشان می‌دهد  

---

## 🧩 افزونهٔ مرورگر

- سایت‌های مزاحم را در سه سطح `always` (همیشه)، `focus` (فقط زمان فوکوس) و `rest` (فقط زمان استراحت) مسدود می‌کند
- حتی با بستن برنامه، سایت‌های `always` از طریق حافظهٔ افزونه مسدود می‌مانند
- صفحهٔ «سپر تمرکز» با تمرین تنفس و میانبرهای مفید، شما را به مسیر کار برمی‌گرداند

---

## 🛠️ فناوری‌ها

- **زبان:** Python 3.12
- **رابط کاربری:** PyQt6
- **تشخیص چهره:** OpenCV (Haar Cascade + DNN)
- **افزونه:** JavaScript (Manifest V3)
- **نمودارها:** Matplotlib
- **بسته‌بندی:** PyInstaller + dpkg

## 📁 ساختار پروژه

```
Mikhkoob/
├── core/              ← منطق اصلی (تایمر، تشخیص حضور، مسدودساز و ...)
├── ui/                ← رابط کاربری (پنجره اصلی، نمودارها، تنظیمات)
├── extension/         ← افزونهٔ مرورگر
├── data/              ← فایل‌های داده (لیست مسدودی، پیام‌ها، پروفایل‌ها)
├── assets/            ← فونت و مدل تشخیص چهره
├── models/            ← مدل DNN
├── build_deb.sh       ← اسکریپت ساخت بستهٔ .deb
└── run.sh             ← اجرای برنامه در محیط توسعه
```
